# Indiv_Results_Holes — Hole-Level Normalized Output

Expands the staging data into one row per player **per hole**, enabling
advanced analytics.

### Responsibilities
- Generate 18 hole-records per player-round  
- Carry par, handicap, score, net-score, and points for each hole  
- Integrate birdie/eagle/double-eagle flags  
- Preserve metadata (week, team, position, opp, side)  

### Downstream Usage
- Hole difficulty modeling  
- Blind-draw optimization  
- Birdie/eagle frequency reporting  
- Visualization tools
