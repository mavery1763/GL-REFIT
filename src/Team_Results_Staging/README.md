# Team_Results_Staging — Canonical Team Staging Table

Transforms Raw team totals into canonical staging format.

### Responsibilities
- Validate team-level data  
- Add `TeamKey` and `MatchKey` identifiers  
- Preserve net, points, and match outcomes  
- Standardize metadata fields  

### Downstream Usage
- Standings  
- Historical reporting  
- Multi-season analytics
